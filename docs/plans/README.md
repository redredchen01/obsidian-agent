## Architecture Context

The Knowledge Graph (_graph.md) generated by IndexManager uses a 4-column format:

```
| Source | Links To | Type | Navigation |
|--------|----------|------|------------|
| [[2026-03-27]] | [[2026-03-28]] | nav | nav-next |
| [[2026-03-27]] | [[2026-03-26]] | nav | nav-prev |
| [[proj-a]]     | [[proj-b]]     | strong | |
```

The nav-prev/nav-next format enables:
- Timeline reconstruction
- Chronological navigation
- Graph tool filtering
- Obsidian timeline plugin integration
