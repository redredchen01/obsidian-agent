## 🔴 已知问题 (优先级)

### P1 - 立即修复（性能影响）

**Issue 1**: `vault.scanNotes()` 重复调用
- **位置**: `src/vault.mjs` 多处
- **影响**: 大库搜索慢 5-10x
- **修复**: 实现 scanNotes 缓存层
- **估计耗时**: 0.5h

**Issue 2**: 聚类重建 O(n) 查找
- **位置**: `src/index-manager.mjs`
- **影响**: 1000+ 笔记库变 O(n²)
- **修复**: 使用 Map 替换数组查找
- **估计耗时**: 1h

### P2 - 下一版本（代码质量）

3. **数值舍入一致性**
